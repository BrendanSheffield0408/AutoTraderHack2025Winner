Built a multi-modal regression to predict "days_to_sell" for auto trader dataset, whilst also assessing impact of the features and utilisation of insights gleemed from analysis in a buisness environment. 

Data included:
- **Geographic Features**: Selling Venue locations (postcode, region, etc)
- **Vehicle Features**: Vehicle brand (Ferrari/ Peugot, etc), Vehicle Condition, Age, Horse-power, Fuel type, etc
- **Advert Features**: Attched Slogans/adverts to the advertisement
